1. Place the jupyter notebook, "ML Final Project Notebook.ipynb" and the training and test data csv files in the same folder.
2. The output CSV, "test_outputs.csv" contains the column name "id_num". Producing this output file via code will result in a blank column name. Please add this column name manually.
3. The index in id_num column in the "test_outputs.csv" currently starts from 1, following the indexing in the training dataset. Producing this output file via code will result in indexing starting from 0.  Please increment all the values in this column manually to start the index from 1.
4. Pre-processing time: Approximated 1 hour.
